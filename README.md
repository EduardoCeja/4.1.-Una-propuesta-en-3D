# Juego 3D en Unity

Proyecto básico desarrollado en Unity como práctica académica para crear un videojuego en 3D.

## Descripción

El juego consiste en controlar un personaje dentro de un escenario 3D, desplazarse por el mapa, evitar obstáculos y recolectar monedas para aumentar la puntuación.

## Funciones principales

* Movimiento del personaje con teclado.
* Cámara en tercera persona que sigue al jugador.
* Sistema de colisiones con obstáculos.
* Recolección de monedas.
* Contador de puntos.
* Reinicio automático si el jugador cae del escenario.

## Controles

* **W / Flecha Arriba:** Avanzar
* **S / Flecha Abajo:** Retroceder
* **A / Flecha Izquierda:** Girar izquierda
* **D / Flecha Derecha:** Girar derecha
* **Espacio:** Saltar

## Tecnologías utilizadas

* Unity
* Lenguaje C#
* Visual Studio Code o Visual Studio
